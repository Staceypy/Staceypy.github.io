Software Process Quiz

2020-11-10 2 min read
Software process is a set of related activities that leads to the production of a software product.

1. Software Specification

2. Software design and implementation

3. Software Validation

1. Development Testing:

Componets, unit testing

2. System Testing:

1. This process is concerned with finding errors that result from unanticipated interactions between components and component interface problems

2. It is also concerned with showing that the system meets its functional and non-functional requirements, and testing the emergent system properties.

3. Acceptance Testing

The system is tested with data supplied by the system customer rather than with simulated test data.

4. Software Evolution

The software is modified to reflect changing customer and market requirements.

QUIZ

1. Use cases are identified using __based requirements elicitation technique

Answer: User interface (UI) analysis is a requirements elicitation method applied by a BA to explore an existing UI and learn how users interact with it.

2. In Scrum, ____________ is a facilitator who arranges daily meetings, tracks the backlog of work to be done, records decisions, measures progress against the backlog, and communicates with customers and management outside of the team.

Answer: Scrum Master

3. Requirement engineering process consists of

  1. Feasibility study
  2. Requirement gathering
  3. Software requirement specification and validation
