os.walk()方法用于通过在目录树中游走输出在目录中的文件名，向上或者向下.

os.walk(top[, topdown=True[, onerror=None[, followlinks=False]]])

参数

  • top -- 是你所要遍历的目录的地址, 返回的是一个三元组(root,dirs,files)
    • root 所指的是当前正在遍历的这个文件夹的本身的地址: 例 ，C:\Users\dell\Desktop\CV\test
    • dirs 是一个 list ，内容是该文件夹中所有的目录的名字(不包括子目录)
    • files 同样是 list , 内容是该文件夹中所有的文件(不包括子目录)
  • topdown: 为 True，则优先遍历 top 目录，否则优先遍历 top 的子目录

实例

    import os
    for root, dirs, files in os.walk(r'C:\Users\dell\Desktop\CV\test', topdown=True):
        for name in files:
            print(os.path.join(root, name))
        print('done')
        for name in dirs:
            print(os.path.join(root, name))

执行以上程序输出结果为：

C:\Users\dell\Desktop\CV\test\Yu_Pei_CV_the_former_one.pdf
C:\Users\dell\Desktop\CV\test\杀生丸.jpg
C:\Users\dell\Desktop\CV\test\犬夜叉.jpg
done
C:\Users\dell\Desktop\CV\test\inner
C:\Users\dell\Desktop\CV\test\inner\conselling form.docx
done

python - os.sep用法

python是跨平台的。
在Windows上，文件的路径分隔符是'\'，在Linux上是'/'。
os.sep根据你所处的平台，自动采用相应的分隔符号。
例：data_dir = os.sep.join(['hello', 'world'])
